# 📚 Study Timer Web

개인 학습 및 활동 기록을 위한 웹 어플리케이션입니다.

## ✨ 주요 기능

*   **공부 시간 기록**: 과목별 공부 시간을 실시간으로 기록하고 관리합니다.
*   **활동(운동/Craft) 기록**: 30분, 60분 등 단위 시간으로 운동 및 Craft 활동을 간편하게 기록합니다.
*   **통계 대시보드**: 일간, 주간, 월간 학습 통계 및 과목별 비교 차트를 제공합니다.
*   **오프라인 동기화**: 인터넷이 끊겨도 로컬에 기록했다가 나중에 서버와 동기화할 수 있습니다.
*   **멀티 스톱워치**: 여러 과목을 동시에 측정할 수 있는 멀티 스톱워치 기능을 제공합니다.

## 🚀 실행 방법

이 프로젝트는 `pnpm`을 사용하여 간편하게 실행할 수 있습니다.

```bash
# 의존성 설치 및 서버 실행
pnpm dev
```

서버는 기본적으로 `http://127.0.0.1:5001`에서 실행됩니다.

## 🔒 보안 및 로그인

이 앱은 **단일 관리자 비밀번호** 방식을 사용합니다.

1.  최초 접속 시 로그인 페이지가 나타납니다.
2.  기본 비밀번호는 **`0000`** 입니다.

### 비밀번호 변경

보안을 위해 비밀번호 변경을 권장합니다. 프로젝트 루트에 `.env` 파일을 생성하고 아래 내용을 추가하세요.

```env
ADMIN_PASSWORD=내비밀번호    
SECRET_KEY=임의의_복잡한_문자열   #사용자가 임의로 생성
```

## 📁 프로젝트 구조

```
stweb/
├── app.py              # 메인 Flask 어플리케이션
├── package.json        # pnpm 스크립트 설정
├── requirements.txt    # Python 라이브러리 목록
├── templates/          # HTML 템플릿
│   ├── index.html      # 메인 대시보드 (통계, 타이머, 활동기록)
│   ├── login.html      # 로그인 페이지
│   ├── multi.html      # 멀티 스톱워치
│   └── history.html    # 상세 기록 내역
└── .env (선택)          # 환경 변수 설정 파일
```
